üîß COMMON PROBLEMS AUR SOLUTIONS:
================================================================================

PROBLEM 1: Browser nahi khul raha
SOLUTION: 
    open_browser.bat


PROBLEM 2: Server nahi chal raha
SOLUTION:
    taskkill /F /IM python.exe
    start_veda.bat


PROBLEM 3: Port already use mein hai
SOLUTION:
    for /f "tokens=5" %a in ('netstat -ano ^| findstr ":8000"') do taskkill /F /PID %a
    start_veda.bat


PROBLEM 4: Page load nahi ho raha
SOLUTION:
    - 5 seconds wait karo
    - F5 dabao (refresh)
    - Logs check karo: type logs\veda_ai.log
